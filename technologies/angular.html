Angular Configurations
  • Create Angular Project
    • ng new project-name
    • ng new project-name --skip-tests (without spec files)
  • Run Angular Project
    • ng serve --open
    • ng s -o
    • ng serve --open --aot --prod
  • Create a component
    • ng g c component_folder/component_name --spec=false --flat=true --skip-tests
  • Create a Module
      ng g m app-routing --flat=true --module=app
  • Create a Directive
    • ng g directive directive_name [options]
  • Create a Service
    • ng g service service_folder/servce_filename
Node Packages
  • Using "as-split" for spillter which can be dragged
    • npm i angular-split
  • Install bootstrap
    • npm install --save @ng-bootstrap/ng-bootstrap
  • Install Amexio
    • npm install amexio-ng-extension@latest --save
  • Install fontawesome
    • npm install --save-dev @fortawesome/fontawesome-free
